# PythonExamples

A collection of 100 Python examples for learners.

## What is included

- 29 original examples in the repository root.
- 71 new examples in `examples/` (`example_030_*.py` to `example_100_*.py`).

## Topic coverage

- Strings and formatting
- Lists, dictionaries, sets, and comprehensions
- Functions, `*args`, `**kwargs`, and closures
- Error handling and custom exceptions
- Files, CSV, JSON, and temporary files
- OOP, dataclasses, properties, class/static methods
- Iterators, generators, decorators, and context managers
- Algorithms (search, sort, primes, recursion)
- Standard library tools (`itertools`, `collections`, `heapq`, `bisect`)
- CLI, logging, environment variables, HTTP, sqlite, asyncio

## Run examples

Run any example directly:

```bash
python3 examples/example_033_list_comprehension.py
```

Or run one of the original files:

```bash
python3 factorial.py
```
